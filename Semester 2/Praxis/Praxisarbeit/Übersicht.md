## 3. Was ist Agentic AI?

Das Kernkapitel.

### Mögliche Inhalte

Definitionen aus der Literatur:

Agentic AI beschreibt KI-Systeme, die:

* Ziele verfolgen
* Entscheidungen treffen
* Aktionen planen
* Werkzeuge verwenden
* ihre Ergebnisse bewerten
* autonom mehrere Schritte ausführen

### Abgrenzung

Vergleich:

| Klassisches LLM       | Agentic AI          |
| --------------------- | ------------------- |
| Antwortet auf Prompts | Verfolgt Ziele      |
| Einzelschritt         | Mehrschrittprozesse |
| Reaktiv               | Proaktiv            |
| Kein Gedächtnis       | Memory möglich      |
| Keine Aktionen        | Kann handeln        |
